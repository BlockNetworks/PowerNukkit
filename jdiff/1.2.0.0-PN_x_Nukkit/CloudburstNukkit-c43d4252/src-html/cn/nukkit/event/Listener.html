003/**
004 * 所有的监听事件的类必须实现的接口。<br>
005 * An interface implemented by all classes that handles events.
006 *
007 * <p>插件要监听事件，需要一个类实现这个接口，在这个类里编写方法来监听。这个类称作<b>监听类</b>。
008 * 监听类中监听事件的方法称作事件的<b>处理器</b>。一个监听类可以包含多个不同的事件处理器。
009 * 实现监听类后，插件需要在插件管理器中注册这个监听类。<br>
010 * If a plugin need to listen events, there must be a class implement this interface. This class is called a <b>listener class</b>.
011 * Methods with specified parameters should be written in order to listen events. This method is called a <b>handler</b>.
012 * One listener class could contain many different handlers.
013 * After implemented the listener class, plugin should register it in plugin manager.</p>
014 *
015 * <p>事件监听器被注册后，Nukkit会在需要监听的事件发生时，使用反射来调用监听类中对应的处理器。<br>
016 * After registered, Nukkit will call the handler in the listener classes by reflection when a event happens.</p>
017 *
018 * <p>这是一个编写监听类和处理器的例子。注意的是，标签{@code @EventHandler}和参数的类型是必需的：<br>
019 * Here is an example for writing a listener class and a handler method.
020 * Note that for the handler, tag {@code @EventHandler} and the parameter is required:</p>
021 * <pre>
022 * public class ExampleListener implements Listener {
023 *    {@code @EventHandler(priority = EventPriority.NORMAL, ignoreCancelled = false)}
024 *     public void onBlockBreak(BlockBreakEvent event) {
025 *          int blockID = event.getBlock().getPackId();
026 *          if (blockID == Block.STONE) {
027 *              event.getPlayer().sendMessage("Oops, my ExampleListener won't let you break a stone!")
028 *              event.setCancelled(true);
029 *          }
030 *     }
031 * }
032 * </pre>
033 *
034 * <p>关于注册监听类，请看：{@link cn.nukkit.plugin.PluginManager#registerEvents}.<br>
035 * For registering listener class, See: {@link cn.nukkit.plugin.PluginManager#registerEvents}.</p>
036 *
037 * <p>关于处理器的优先级和处理器是否忽略被取消的事件，请看：{@link EventHandler}.<br>
038 * For the priority of handler and whether the handler ignore cancelled events or not, See: {@link EventHandler}.</p>
039 *
040 * @author Unknown(code) @ Nukkit Project
041 * @author 粉鞋大妈(javadoc) @ Nukkit Project
042 * @see cn.nukkit.event.Event
043 * @since Nukkit 1.0 | Nukkit API 1.0.0
044 */
